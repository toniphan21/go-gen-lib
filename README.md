# go-gen-lib

A Go code generation library that provides utilities for creating, managing, and organizing generated source files.

## Features

- **FileManager** - Manage generated Go source files with configurable header comments
- **Setup utilities** - Load Go packages and write source files to disk

## Installation

```bash
go get nhatp.com/go/gen-lib
```

## Quick Start

```go
package main

import (
    "nhatp.com/go/gen-lib"
)

func main() {
    // Create a FileManager with custom options
    fm := genlib.NewFileManager(
        "/my/project",
        genlib.WithBinaryName("my-generator"),
        genlib.WithVersion("v1.0.0"),
    )
	
    // now 
    generator := NewYourGenerator(fm)
}
```

## FileManager Options

```go
// Set the generator binary name (default: "generator")
WithBinaryName("mygen")

// Set the version string (default: "dev")
WithVersion("v1.2.3")

// Customize header comment template
WithHeaderCommentTemplate("Generated by {binaryName} {version}")
```

## jennifer Utilities

| Function                   | Description                                 |
|----------------------------|---------------------------------------------|
| `TypeToJenCode`            | Convert go/types to jen.Code                |
| `VarToJenCode`             | Convert function params/results to jen.Code |
| `TypeSimpleName`           | Get simple type string representation       |
| `ZeroValueOfType`          | Generate zero values for types              |
| `WrapComment` / `WrapText` | Text wrapping utilities                     |
